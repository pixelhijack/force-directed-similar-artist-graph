Concept | Last.fm similar artist discovery with D3.js force directed graph modelling

  1. Search artist, pull Last.fm API for similar artists. Last.fm will give you the similarity correlation ratio as "match" in the JSON.
  2. Render a D3.js force directed graph layout by passing the match data. Implement drag functionality with corresponding physical modelling changes on every tick event of D3
  3. Use mock data first to test your model.
  4. Listen for view interactions, update model and re-render the force layout. Clicking on an artist should pull Last.fm again for similar artists of that node and populate the graph with new model data
  5. When adding new nodes, loop through previous ones and merge new elements, linking the edges on common similarities.
